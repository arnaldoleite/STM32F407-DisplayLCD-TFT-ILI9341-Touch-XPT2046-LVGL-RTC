# Firmware for embedded systems using the STM32 F4VE black board with STM32F407 microcontroller + ILI9341 TFT LCD display (320x240) + XPT2046 touchpad + LVGL + RTC

![STM32F4VE](https://stm32-base.org/assets/img/boards/STM32F407VET6_STM32_F4VE_V2.0-1.jpg)

Development of a graphical interface on an STM32F407 microcontroller from STM32 using an ILI9341 TFT LCD in parallel mode (FSMC) and SPI resistive touch, using the LVGL framework.

Graphical interface developed using EEZ Studio software.
